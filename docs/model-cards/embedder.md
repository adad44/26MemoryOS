# Model Card: MemoryOS Embedder

## Purpose

Encode captures and queries into vectors for semantic search.

## Base Model

Default:

```text
sentence-transformers/all-MiniLM-L6-v2
```

## Fine-Tuning Data

Generated by:

```sh
python3 ml/train/generate_pairs.py
```

Pair labels:

- `1.0`: same app/day/context positive pair
- `0.0`: different app/context negative pair

## Training

```sh
python3 ml/train/finetune_embedder.py
```

Output:

```text
ml/models/memoryos-embedder/
```

## Evaluation

Manual evaluation:

- Search for remembered topics.
- Confirm top result correctness.
- Target checkpoint: correct top result in 7 out of 10 manual queries.

## Limitations

- Automatically generated pairs are noisy.
- Fine-tuning needs enough real captures to improve over the base model.
- Private personal data should not be uploaded to external training services.
